ros智能小车建图定位导航

---

## 项目简介

本项目旨在实现一套基于 **3D 激光雷达 (LiDAR)** 的移动机器人导航与定位系统，结合 **Fast-LIO** 高精度定位算法和 **ROS Navigation Stack** 的 2D 导航，实现机器人在复杂室内环境下的自主移动。

**特色亮点：**

* 支持 **Fast-LIO 实时 3D 定位**，在动态环境中仍保持稳定性
* 利用 **3D 激光转 2D 激光** 与 ROS Navigation Stack 对接，实现高效路径规划

---

## 功能特性

1. **高精度 3D 定位**

   * 使用 Fast-LIO 完成全局地图匹配
   * 支持 IMU 融合，减少漂移

2. **二维导航与避障**

   * 利用 costmap_2d + move_base 完成全向移动与动态障碍避让
   * 可自定义局部规划器（DWA / TEB）参数
